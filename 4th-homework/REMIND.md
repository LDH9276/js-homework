# 16기 이동헌 JavaScript 4주차 과제

## 과제 수행 과정
- 처음에 루프를 실습에 사용하기 위해 객체화를 했습니다.
- 우선 시맨틱 코드로 짠 후에 백틱으로 삼항연사자를 통해 처음 있는 인덱스에 클래스를 주고 초기화하도록 하였고
- 각 안의 내용 클래스와, 접근성을 위해 `arid-selected`와 `hidden`을 사용했습니다. (추후 애니메이션이 들어가면 클래스 제어로 수정해야 될 듯 싶어요.)

## 테스트 코드 작성
``` bash
bun dev
```

## 과제 코드 리뷰
- 컴포넌트 방식을 생각하기 위해 상단 버튼은 리액트 처럼 객채화를 하여 for 문으로 해보았습니다.
- 다만 랜더링 후 이벤트 리스너 부여가 안되어 보니 처음 렌더링이 끝나고 변수를 지정해야 되는 것을 알았습니다.
- 객체화 과정은 따로 빼서 불러올 수 있습니다만, 나중에 `import export`를 배우고서 쉽게 쓸 수 있을 것 같습니다.
